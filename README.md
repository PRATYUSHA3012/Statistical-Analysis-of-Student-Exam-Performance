# Statistical-Analysis-of-Student-Exam-Performance


| **Business Question**                                | **Analysis**           | **Key Finding**                                                                        |
| ---------------------------------------------------- | ---------------------- | -------------------------------------------------------------------------------------- |
| Does test preparation improve student performance?   | Independent t-test     | Students who completed the test preparation course achieved higher average scores.     |
| Does parental education affect academic performance? | One-Way ANOVA          | Student performance varied across different parental education levels.                 |
| Are there unusual student scores?                    | IQR Outlier Detection  | A few extreme score values were identified for further investigation.                  |
| Are reading and writing scores related?              | Pearson Correlation    | Reading and writing scores showed a strong positive relationship.                      |
| What is the overall performance distribution?        | Descriptive Statistics | The dataset showed the overall trend, variability, and distribution of student scores. |
